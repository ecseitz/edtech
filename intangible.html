Intangible

The Intangible project is the virtual version of a physical tool that I created. It demonstrates the finished product that this tool would become. It was written all in JavaScript/HTML/CSS. Below, I will describe the original physical tool, but the virtual version works the same way. Click here for specific instrustions on using the virtual tool.

Inspiration

The Intangible project started out as a physical tool to help students learn chemistry. I was inspired by the classic activity of drawing Lewis dot diagrams to study how electrons are arranged when atoms bond to form molecules. I had to draw these diagrams in high school chemistry, as do many other students. I wanted to design a tool that would make this highly conceptual activity more tangible - hence the name "Intangible."

Beyond electron configurations, molecules also have specific bond angles. I used flex sensors to mark where the bonds between molecules occur, and these sensors can read approximately the angle at which they are bent. Students using the tool can therefore configure the electron LEDs and then bend the bonds to the correct angle in order to model a specific molecule.

Physical design

I designed round "atoms" that included LEDs in the place of electrons. Each atom has a battery and the central atom has a LilyPad Arduino running the tool's code. In the following example, the blue central atom is Oxygen, and each of the red atoms are Hydrogen. This is the water molecule.

The user turns on the lights and bends the arms, and if they are correct an LED in the central atom blinks. For water, there should be six LEDs on for Oxygen and one for each Hydrogen. The angle between the bonds is ~105Â°. It looks like the diagram above.

Binary math

After designing this tool for a class, the professor challenged me to come up with an alternate application. The on/off nature of the LEDs reminded me of the 0/1 of binary numbers, and so I recoded the tool to help students practice binary math.

For math, I designated each light to represent a different power of 2, and the arrangement follows a couterclockwise circle. These powers of 2 are clearly marked in the virtual version. The user turns on various lights for the two smaller atoms, performs some mathematical function on those two numbers, and then lights up the central atom with the result. There are multiple modes, including multiplication and addition. When the user then bends the bonds, it is a signal for the LilyPad to check the result. If the math is correct, then it flashes an LED.
