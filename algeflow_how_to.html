Buttons

Submit

Checks the order of the blocks in the canvas area.

Next

Moves onto the next problem after a problem is completed successfully. For now, this button must be pressed to start the tool upon loading the page.

Blocks

Blocks are draggable and droppable between the block area and the canvas. They include simple math and more complex algebraic concepts. When a function must be performed on both sides of an equation, the block reminds the users with the intention of ingraining the "both sides" idea in their heads.

Equation

The equation for a problem appears above the canvas area.
